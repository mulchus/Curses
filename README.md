# Скрипт "Игра про космос"


## Настройки

Для запуска у вас уже должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой 
```
pip install -r requirements.txt
```


## Запуск
- Запустите скрипт командой 
```
python main.py
```
